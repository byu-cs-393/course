## Course at a glance

Students build the confidence and skills to solve technical-interview–style problems and to
approach their career search well. Practice centers on LeetCode-style problems, mock
interviews with real professionals, topic exams, and a comprehensive final.

See the full [**syllabus**](syllabus.md).

## What changed for Fall 2026

1. **Moved off Learning Suite self-report quizzes → Canvas.** Previously students
   self-reported completion via Learning Suite quizzes (a true/false quiz for a mock
   interview; a form naming who they tested with for a topic exam). These are now Canvas
   assignments that capture the report content directly.
2. **Topic exams: 8 → 12.** Each of the four topics now has **three** exams — an Online
   Assessment, a new **Performance Exam** (practiceable ahead of time), and a Live
   Interview. See [`topic-exams/`](topic-exams/).
3. **Two submission paths, extension always optional.** Topic-exam completion can flow in
   through [Jack Leonard's Chrome extension](https://github.com/byu-cs-393/extension) *or*
   through a plain, manually-graded Canvas assignment. The course works fully **without**
   the extension.

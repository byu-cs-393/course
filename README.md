## Topic-exam submission: extension vs. manual

Students demonstrate a topic exam live with a TA/instructor. Recording that it happened
can go two ways:

- **Chrome extension (optional):** [Jack Leonard's extension](https://github.com/byu-cs-393/extension)
  lets a TA approve a completed exam and push the result straight into Canvas.
- **Manual (always supported):** a per-exam Canvas assignment the student submits
  (what they did, who they tested with), graded manually by the TA/instructor.

The extension is a convenience layer — **never required**. See
[`topic-exams/README.md`](topic-exams/README.md) for the full process.

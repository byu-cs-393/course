# Amazing Project

Build something real and make it public. (Strongly recommended: use a
[claude.ai](https://claude.ai) subscription to help you build.)

There are **three opportunities**, and you can do **each once for credit** — but do
them your whole career for growth!

1. **Community / World** — helps the community or the world in some not-for-profit way.
2. **Personal** — a personal project, perhaps only useful to you.
3. **Paid** — earns money. Show proof you were paid, and it must be **freelance** (not part
   of the responsibilities of hourly or salaried employment). This one needn't be a public
   repo, but it must be public enough for a TA to manually approve.

## What makes a project "Amazing"

- **Public GitHub repo** with meaningful value that represents significant work — **target
  20+ hours** (even if AI-assisted).
- **A time log.** Each feature commit says **how much time that feature took**, so the effort
  is visible in the commit history.
- **A README.** The **root README embeds a 1–3 minute video** that explains why the project is
  interesting and **demos** the functionality.
- The repo/README **mentions you** and links to something like your **LinkedIn** profile.

## Submitting

Submit your **repo URL** to the matching Canvas assignment (Community, Personal, or Paid). A TA
reviews it against the requirements above.

# Topic Exam (Performance) — Graphs

> **New for Fall 2026.** The performance exam is the second of the three topic exams. Unlike
> the live interview, you are allowed to **practice the question ahead of time** and then
> perform it. See [Topic Exams](../README.md) for how the three exams fit together.

## The question

**[Course Schedule IV](https://leetcode.com/problems/course-schedule-iv/)**

Given prerequisite pairs, answer queries about whether one course is a prerequisite
(directly or indirectly) of another. Exercises directed-graph reachability — DFS/BFS from
each node, or a transitive-closure approach.

## How it works

1. Practice the question above ahead of time.
2. Perform it for a TA or the instructor.
3. Record completion in Canvas — via the Chrome extension, or the
   [manual submission template](../submission-template.md).

## Learning outcomes

This exam targets the same outcomes as the [online assessment](online-assessment.md):

- Implement solutions requiring DFS.
- Implement solutions requiring BFS.
- Implement solutions requiring cycle detection in a graph.

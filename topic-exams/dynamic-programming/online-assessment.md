## Learning objectives

Implement DP solutions using DFS with memoization (boolean, sum, min/max)

Implement counting solutions where a formula is used to increment a total count at each iteration (e.g. N(N-1)/2 for subsequences)

Implement solutions that use a prefix or postfix to make them more efficient

Implement bottom up solutions that build up from some base case iteratively
